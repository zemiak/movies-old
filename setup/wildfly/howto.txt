Wildfly 8 AS Setup

PostgreSQL Driver

Taken from Stack Overflow

Install the driver

Create path “{WILDFLY}/modules/system/layers/base/org/postgresql/main/”. Put there a file called “module.xml” with the following content:

<?xml version="1.0" encoding="UTF-8"?>
<module xmlns="urn:jboss:module:1.1" name="org.postgresql">
    <resources>
	<resource-root path="postgresql-9.3-1101.jdbc41.jar"/>
    </resources>
    <dependencies>
	<module name="javax.api"/>
	<module name="javax.transaction.api"/>
	<module name="javax.servlet.api" optional="true"/>
    </dependencies>
</module>

Download there also the mentioned driver:

“curl http://jdbc.postgresql.org/download/postgresql-9.3-1101.jdbc41.jar >postgresql-9.3-1101.jdbc41.jar”

Register the driver

Run the command “{WILDFLY}/bin/jboss-cli.sh —connect”
And then type on one line:

/subsystem=datasources/jdbc-driver=postgresql-driver:add(driver-name=postgresql, driver-class-name=org.postgresql.Driver, driver-module-name=org.postgresql)

Output should be:

{"outcome" => “success"}
